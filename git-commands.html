GIT Commands

  • git clone <url>
  • git add <filename> (for adding all files git add .)
  • git status
  • git commit -m "message" (for adding and commiting at the same time git commit -am "message")
  • git push (when updated version is stored on local)
  • git pull (when updated version is stored on server i.e github)
  • git reset --hard <commit> git reset --hard origin/master
